# Frontend

UI layer for the whiteboard.

## Responsibilities
- HTML/CSS layout
- Hermes desktop integration (UI pane, commands)
- Tool palette UI
- Presence indicators (cursors, selections)
- Export (PNG, SVG, JSON)

## Integration Options
- Hermes desktop plugin (UI pane + command)
- Standalone web app (opens in browser)
- Both (plugin launches web UI)
