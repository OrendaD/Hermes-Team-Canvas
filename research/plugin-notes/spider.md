# Spider

**Pattern:** Node-as-Conversation + Knowledge Graph
**Repo:** 111pointer111/spider
**Version:** v0.1.10
**Status:** Very new

## Why It Matters

Most visually sophisticated agent-canvas integration. React Flow canvas with auto-layout and persistent positioning.

## How It Works

- Tab-branching: select text in AI response → Tab → child node
- Interactive knowledge graph via React Flow (full-tab canvas)
- Auto-layout via Dagre algorithm
- Drag nodes — positions persist
- Export: Markdown, Canvas, Mermaid mindmap
- Multi-provider: ChatGPT, Claude, DeepSeek, any OpenAI-compatible

## Agent-Canvas Relationship

Treats canvas as first-class visualization surface. Agent generates nodes, then steps back. Auto-layout + persistent positioning means output is spatially meaningful.

## Building Block for Pattern 5

React Flow canvas, auto-layout with persistent positioning, node collapse/expand. The UI model to beat for Pattern 5.
