# Advanced Canvas

**Pattern:** Event-Driven (Primary)
**Repo:** developer-mike/obsidian-advanced-canvas
**Version:** v6.5.0
**Downloads:** ~1,400

## Why It Matters

The most important canvas plugin for Hermes. Its **events API** is the bridge between human canvas actions and agent processing.

## Key Capabilities

- **Canvas events API** — exposes node created/moved/modified/deleted events to other plugins
- **Frontmatter support** — canvas nodes carry metadata
- **Presentation mode** — slide-like navigation (1200×675 groups)
- **Flowchart shapes** — diamond, hexagon, cloud
- **Portals** — embed canvases within canvases
- **Node templates and styles**

## Agent Relevance

The events API is THE integration point. When a human creates a node on the canvas, an agent plugin could receive the event and respond. This is how Pattern 5 starts.
