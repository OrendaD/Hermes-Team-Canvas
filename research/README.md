## How to Add Research

1. Check if a document for your topic already exists
2. If yes, add your findings to it (keep sources cited)
3. If no, create a new file following the naming convention: `topic-name.md`
4. Update this index
5. Submit a PR

## Source Attribution

Always cite the source URL and date of research. Distinguish opinion from fact. Link to related files.
