# Canvas Engine

Core canvas rendering and interaction logic.

## Responsibilities
- Render shapes, drawings, text, images
- Handle input (mouse, touch, pen)
- Manage viewport (pan, zoom)
- Object selection and manipulation
- Tool palette (pen, shapes, text, eraser, etc.)

## Rendering Approach (TBD)
- Canvas2D for drawing performance
- DOM overlay for UI elements (toolbars, menus)
- Hybrid approach for best of both worlds
