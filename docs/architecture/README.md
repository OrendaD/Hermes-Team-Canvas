## How to Write an ADR
1. Use `templates/adr.md` as your starting template
2. Number sequentially: `001-<slug>.md`, `002-<slug>.md`, etc.
3. Place in this directory
4. Add a row to the index above
5. Update status in the ADR header as decisions are made
