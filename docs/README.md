# Documentation

## How to Contribute
1. Pick a domain or create a new one under `research/`
2. Use `templates/research-dive.md` as your starting template
3. For decisions, use `templates/adr.md` and place in `architecture/`
4. Diagrams go in `diagrams/` as SVG
5. Reference extractions go in `references/` with provenance
